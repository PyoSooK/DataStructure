# queue ADT

## data
- 정수(int)형 배열

## operation
- init_queue() : 큐를 초기화한다.
- is_empty() : 큐가 비어있으면 True, 아니면 False를 반환한다.
- is_full() : 큐가 가득차있으면 True, 아니면 False를 반환한다.
- size() : 큐 내의 모든 요소들의 개수들을 반환한다.
- enqueue(x) : 주어진 요소 x를 큐 후단에 삽입한다.
- dequeue() : 큐 전단에 있는 요소를 삭제하고 반환한다.
- peek() : 큐 전단에 있는 요소를 삭제하지 않고 반환한다.
